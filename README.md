# Analysis of my Goodreads data

This repository contains an exploratory data analysis of my Goodreads data, implemented entirely in R. The project includes:

- **Data wrangling and querying**: access and process data from APIs using SPARQL and JSON.  
- **Exploratory data analysis and visualization**: dataviz using `ggplot2`.
